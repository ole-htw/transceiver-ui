# Map-Feature Checkliste

- [ ] Mission ohne `map_config` funktioniert unverändert.
- [ ] Mission mit gültiger `map_config` lädt ohne Fehler.
- [ ] Waypoints sind sichtbar und korrekt platziert.
- [ ] Live-Marker aktualisiert sich während Navigation.
- [ ] Fehlende Karten-Datei führt zu verständlicher UI-Meldung statt Exception.
